Cursos Preferidos

Ingeniería de Software I

En el curso de Ingeniería de Software I con el profesor Alan Calderón, aprendimos a trabajar utilizando la metodología Scrum. Además, empezamos a aprender sobre el desarrollo de aplicaciones en Android. Elaboramos un juego llamado "Cazando al Wumpus" que utiliza una librería sencilla de realidad aumentada llamada BeyondAR.

Ingeniería de Software II

El curso de Ingeniería de Software II también fué impartido por el profesor Alan Calderón. Aquí seguimos trabajando con la metodología Scrum y desarrollando en Android. Elaboramos una aplicación para la evacuación de las comunidades en riesgo en la Cuenca del Río Uruca en Santa Ana.

Bases de Datos I

En este curso mi profesor fué Carlos Lobo. Aprendimos sobre bases de datos relacionales SQL, específicamente utilizamos el motor de bases datos SQL Server. Desarrollamos una pequeña aplicación de punto de venta como proyecto del curso.

Bases de Datos II

El profesor de este curso fué Luis Gustavo Esquivel. Primero aprendimos sobre el gestor de bases de datos Oracle. Luego empezamos a investigar y aprender sobre bases de datos NoSQL como Cassandra, CouchDB y MongoDB. Mi grupo de trabajo se enfocó en Cassandra, pero en el futuro me gustaría aprender sobre MongoDB.

Finalmente, estos son dos cursos que estoy llevando en este último semestre para terminar el bachillerato, espero aprender mucho en ellos para que sean parte de mis cursos preferidos, ya que ambos tratan temas que me interesan mucho:

Desarrollo de Aplicaciones Web

En este curso, impartido por el profesor Braulio Solano estamos aprendiendo a desarrollar aplicaciones y páginas web. Hasta el momento hemos aprendido sobre diseño, html, css y el framework front-end Foundation 6. En la primera tarea implementamos un servidor HTTP básico en Java. Este sitio web personal es otra de las tareas.

Creación de Videojuegos

Este curso es impartido por el profesor Markus Eger, estamos aprendiendo sobre el proceso para desarrollar videojuegos en Unity. El juego que mi grupo va a desarrollar en el semestre se llama "Boltblade", un guerrero que tiene poderes eléctricos.
