Acerca de este sitio

El objetivo de este sitio fué iniciar a aprender las bases del desarrollo web. Es una tarea para el curso Desarrollo de Aplicaciones Web de la Universidad de Costa Rica. Durante el desarrollo de este sitio se practicó con el uso de HTML5, CSS3 y el framework front-end Foundation 6, usado para que el sitio sea responsive, es decir, se adapta a pantallas de computadoras, tabletas y teléfonos inteligentes. El contenido del sitio está relacionado con mi vida personal.
